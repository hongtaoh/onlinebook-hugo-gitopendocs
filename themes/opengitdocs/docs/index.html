An open source static site solution for documentation websites.

Use Your Existing Repository

You can use OpenGitDocs with your current repository and keep your code and documentation all in one place. OpenGitDocs works with static sites hosted on GitHub Pages, GitLab Pages, or Bitbucket Cloud.

Automated Continuous Delivery

A seamless site-generation experience makes it easy to automate your documentation site updates with GitHub Actions or GitLab CI/CD.
